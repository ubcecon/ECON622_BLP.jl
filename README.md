This project uses the random coefficients demand model to explain some coding ideas.

- [docs/blp.qmd](docs/blpq.qmd)  has an initial draft of the share equation and code for various ways to integrate out the random coefficients. Use Quarto to render it into reveal.js slides.
- [src/integrate.jl](src/integrate.jl) contains the integration methods
- [src/blp.jl](src/blp.jl) contains the main BLP project module
